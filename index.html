FlockParty.Games

View Site


FlockParty.Games is a self-discovery gaming platform focused primarily on conversation games. Much like it's predecessor, BreadTrails, FlockParty gathers user information to predict best possible matches in careers and education. But the UI for FlockParty better leverages the social connectivity of gaming and pop-culture to encourage user engagement, allowing for the AI to adapt faster. Imagine is Jackbox.tv meets CollegeBoard.

The front-end was designed with HTML5, CSS3, JavaScipt, Bootstrap, Adobe Illustrator, and Photoshop. The back-end is currently being developed using Java and SQL, and is scheduled to launch Summer 2017.

BreadTrails

BreadTrails is an interactive self-discovery web platform that puts students in control of their educational and professional development. Using labor and education statistics provided by the Bureau of Labor Statistics and National Center for Education Statistics, I was able to create a unique algorithm that links personality traits of users to their optimal career and educational matches.

An MVP was built using Microsoft Excel, Microsoft Access, Photoshop, and UXPin in Spring 2016. This prototype was used in over 100 hours of user and market research to refine and validate the algorithm's effectiveness at three metro Atlanta high schools. BreadTrails' algorithmic IP and documented research lays the foundation of FlockParty.Games - expected to launch in 2017.

Library/Bookstore

This responsive library website was designed using ColdFusion, JavaScript, Bootstrap, CSS3, HTML5, and SQL. The project features encrypted user passwords for safe and secure user login. Public users have access to search functionality, genre browsing, and are able to interact with the site's blog. A management section allows privileged users to update the site with ease, giving admins the ability to manage inventory, create new pages using WYSIWYG, and maintain site images.
